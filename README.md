# Scrapping Basico

Repositorio para el ejercicio de web scraping basico en clase.

## Requisitos
- Python 3.x

## Preparacion del entorno
```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -U pip
pip install -r requirements.txt
```

## Notebooks
- `202602_Webscrapping_basico_1.ipynb`
- `202602_Webscrapping_basico_2.ipynb`

## Uso en VS Code
- Selecciona el interprete de `.venv`.
- Abre los notebooks y ejecuta las celdas.
